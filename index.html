How does JavaScript works?

JavaScript includes extra funtionalities in a website by linking with html from a different file with extension "js" or from inside the "script" tag inside html file. Javascript can do only one command at a time. It works on the user machine and react according to the user behaviours. When user runs a js code from the browser by clicking, pressing etc js code runs in the browser engine and converts it into machine code. In that time computer can do their works. According to the js code it can change html file, fetch data from server or api, modify server data, play audio/video and many more.

Difference between Local Storage And Session Storage.

Local Storage : Local storage stores a large amount of data on the browser and it doesn't get deleted or clear after closing the tab or closing the browser and reopening, the storage is saved. As an example use can say that Youtube is a good example of local storage because it remember our data.

Session Storage : Session storage also stores a large amount of data in the browser, but it is deleted after closing the tab or after closing the browser, and it will open with a new session whenever you want to open it.As an example we can say that Facebook is a good example of session storage because after using if you close it you are not able to find it easily.
This is the main difference between local storage and session storage.
